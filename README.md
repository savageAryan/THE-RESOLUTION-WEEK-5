# THE MUMMBLE DINOSAUR
This game is made in godot engine for hackclub resolution game dev path week 5, Game has a hungry dinosaur that eats words and you have to remember the words that the dinosaur eats and then select them at the end of the game,
## CONTROLS-
* A - LEFT
* D - RIGHT
* SPACE = JUMP
